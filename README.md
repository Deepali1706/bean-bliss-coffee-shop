# ☕ Bean Bliss

A modern and responsive Coffee Shop Landing Page built using **HTML5** and **CSS3**.

## 📖 About

Bean Bliss is a clean and elegant coffee shop landing page that showcases a café's services, menu, gallery, customer reviews, and contact information. The project focuses on responsive design, modern UI, and clean code organization.

## ✨ Features

- Responsive Navigation Bar
- Hero Section
- About Section
- Coffee Menu
- Image Gallery
- Customer Reviews
- Contact Section
- Footer
- Smooth Scrolling
- CSS Animations
- Responsive Design

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Google Fonts
- Font Awesome
